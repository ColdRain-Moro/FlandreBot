## 模块

### 来张芙兰

> 口令: 来张芙兰
> 
> 功能概述: 从Pixiv上以关键词`フランドール･スカーレット 1000users入り`搜索并从结果中随机抽取一张图发出
> 
> 注意事项:
> - R18图片会在发出后三十秒撤回
> - 使用该功能需要配置apiKey, 前往 https://www.acgmx.com/ 申请

### 执行代码

> 口令: runJs (代码)
> 
> 功能概述: 执行JS代码 (支持部分ES6特性)
> 
> 注意事项:
> - 执行代码超时时间为5秒，超过5s执行脚本代码的线程将被强制杀死
